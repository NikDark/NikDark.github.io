Каскадные таблицы стилей

При оформлении HTML-документов приходится по нескольку раз прописывать однотипные тэги. Использование каскадных таблиц стилей (CSS) позволяет значительно упростить эту работу. Поддержка стандарта каскадных таблиц стилейи их программируемых элементов появилась начиная с версии 4.0 и Netscape, и Internet Explorer

Cуществует три основных варианта использования таблиц стилей.

Вариант первый

Описание стиля располагается непосредственно внутри тэга элемента, который Вы описываете. Это делается с помощью параметра STYLE. Этот метод нежелателен, понятно поему: он приводит к потере одного из основных преимуществ CSS -
